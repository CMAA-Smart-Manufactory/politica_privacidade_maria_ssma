Política de Privacidade

Este aplicativo foi desenvolvido para realizar a leitura de códigos QR e consultar informações no serviço Supabase. Não coletamos, armazenamos ou compartilhamos dados pessoais, imagens ou vídeos dos usuários.

Uso da Câmera
O acesso à câmera do dispositivo é utilizado exclusivamente para a leitura de códigos QR. Nenhuma imagem ou vídeo capturado pela câmera é armazenado ou enviado para nossos servidores.

Dados Consultados
As informações obtidas a partir da leitura do QR Code são utilizadas apenas para realizar consultas no Supabase e exibir os resultados ao usuário. Não mantemos registros dessas consultas nem compartilhamos dados com terceiros.

Compartilhamento de Informações
Este aplicativo não compartilha informações com terceiros. Todo processamento ocorre em tempo real e os dados não são retidos após a consulta.

Consentimento
O usuário pode negar o acesso à câmera nas configurações do dispositivo. Nesse caso, a funcionalidade de leitura de QR Codes ficará indisponível.

Atualizações
Esta política pode ser atualizada para refletir melhorias ou mudanças no aplicativo. Recomendamos que os usuários revisem periodicamente esta página.
